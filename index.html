📖 Инструкция по работе с порталом
🎯 Цель портала

Портал предназначен для еженедельного сбора данных об уволенных сотрудниках в каждом магазине. Данные использовались для расчета показателей текучки и наполненности штата.

🔐 Шаг 1 — Вход в портал
  • Открыть файл portal_directors.html в браузере
  • В поле «Тип входа» выберите «Директор магазина»
  • В поле «Ваш магазин» выберите свой магазин из списка
  • Введите пароль (выдаётся администратором)
  • Нажмите «Войти на портал →»
👀 Шаг 2 — проверьте список сотрудников
  • После входа вы покажете таблицу всем сотрудникам вашего магазина.
  • Вверху отображается статистика: всего сотрудников, работает, уволено
  • Проверьте список — все ли сотрудники актуальны
✏️ Шаг 3 — Внесение данных об уволенных

Для каждого уволенного сотрудника необходимо:

1
В столбце «Статус» нажмите на выпадающий список напротив сотрудника и выберите «Уволен»
2
В столбце «Данные представления» указывают точную дату представления (формат: ДД.ММ.ГГГГ)
⚠️ Важно: поле даты активируется только после выбора запроса «Уволен». Без даты отчёта считается неполным.
📤 Шаг 4 — Отправить отчёт

После заполнения данных нажмите кнопку из кнопок:

✅
«Без изменений» — нажмите, если за текущий период представлений не было. Администратор увидит, что вы зашли и подтвердили отсутствие изменений.
📤
«Отправить отчёт» — нажмите, если вы внесете данные об уволенных сотрудниках. Отчёт будет отправлен администратору.
💾
«Сохранить данные» — сохраняет введенные данные без отправки. Используйте, если хотите продолжить заполнение позже.
💡 Полезно знать
  • Данные сохраняются автоматически при каждой стойке — вы не потеряете их при случайном закрытии
  • При следующем входе все ранее введенные данные будут восстановлены
  • Вы можете видеть только свой магазин
  • Портал работает в любом браузере (Chrome, Edge, Firefox)
  • По вопросам обращения к директору подразделения или в отдел кадров
🕐 Срок сдачи отчёта
Отчёт необходимо заполнять 3 раза в месяц — 1-го, 10-го и 20-го числа каждого месяца до 12:00. При наличии вопросов — связывайтесь с администратором заранее.
