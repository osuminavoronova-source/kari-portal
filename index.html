📖 Инструкция по работе с порталом
🎯 Цель портала

Портал предназначен для еженедельного сбора данных об уволенных сотрудниках по каждому магазину. Данные используются для расчёта показателей текучки и наполненности штата.

🔐 Шаг 1 — Вход в портал
  • Откройте файл portal_directors.html в браузере
  • В поле «Тип входа» выберите «Директор магазина»
  • В поле «Ваш магазин» выберите свой магазин из списка
  • Введите пароль (выдаётся администратором)
  • Нажмите «Войти в портал →»
👀 Шаг 2 — Проверьте список сотрудников
  • После входа вы увидите таблицу со всеми сотрудниками вашего магазина
  • Вверху отображается статистика: всего сотрудников, работает, уволено
  • Проверьте список — все ли сотрудники актуальны
✏️ Шаг 3 — Внесите данные об уволенных

Для каждого уволенного сотрудника необходимо:

1
В столбце «Статус» нажмите на выпадающий список напротив сотрудника и выберите «Уволен»
2
В столбце «Дата увольнения» укажите точную дату увольнения (формат: ДД.ММ.ГГГГ)
⚠️ Важно: поле даты активируется только после выбора статуса «Уволен». Без даты отчёт считается неполным.
📤 Шаг 4 — Отправьте отчёт

После заполнения данных нажмите одну из кнопок:

✅
«Без изменений» — нажмите, если за текущий период увольнений не было. Администратор увидит, что вы заходили и подтвердили отсутствие изменений.
📤
«Отправить отчёт» — нажмите, если вы внесли данные об уволенных сотрудниках. Отчёт будет отправлен администратору.
💾
«Сохранить данные» — сохраняет введённые данные без отправки. Используйте, если хотите продолжить заполнение позже.
💡 Полезно знать
  • Данные сохраняются автоматически при каждом изменении — вы не потеряете их при случайном закрытии
  • При следующем входе все ранее введённые данные будут восстановлены
  • Вы можете видеть только свой магазин
  • Портал работает в любом браузере (Chrome, Edge, Firefox)
  • По вопросам обращайтесь к своему директору подразделения или в HR
🕐 Срок сдачи отчёта
Отчёт необходимо заполнять 3 раза в месяц — 1-го, 10-го и 20-го числа каждого месяца до 12:00. При наличии вопросов — связывайтесь с администратором заранее.
